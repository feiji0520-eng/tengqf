# 方鸟冲冲冲（微信小游戏）

一个简化版的“方鸟冲冲冲/Flappy Bird”微信小游戏示例，点击屏幕让小鸟飞行，躲避管道并获得分数。

## 运行方式

1. 打开微信开发者工具。
2. 选择“导入项目”，目录选择本仓库根目录。
3. `appid` 可使用游客模式 `touristappid`。
4. 启动后点击屏幕开始游戏。

## 玩法

- 点击/触摸屏幕，小鸟会上升。
- 撞到管道或上下边界则游戏结束。
- 通过管道会累积分数。
- 主页点击“查看排行”按钮可查看微信好友排行榜（需要在微信开发者工具中启用开放数据域）。
- 主页可选择难度等级，难度越高管道间隙更小、速度更快。
- 游戏结束后会展示微信好友排行榜（需要在微信开发者工具中启用开放数据域）。

## 主要文件

- `game.js`：游戏主逻辑（物理、渲染、碰撞、得分）。
- `game.json`：小游戏配置。
- `project.config.json`：微信开发者工具项目配置。
- `open-data/index.js`：好友排行榜渲染逻辑（开放数据域）。
